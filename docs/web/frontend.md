# 前端服务器的安装配置

前端服务器主要作用是静态界面的展示，所以需要依靠 Nginx 来进行处理。

这里的 Nginx 作用不同于反向代理中 Nginx 的配置，可以精简很多无用的功能，例如：

- [ ] Lua 模块
- [ ] SSL 证书卸载
