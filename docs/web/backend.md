# 后端服务器的安装配置

后端服务器主要用于运行 Java 应用程序，主要需要安装配置的软件应用有：

- [ ] JDK 8 （Java 服务所需运行时）
- [ ] Supervisord （Java 程序管理软件）
- [ ] Nginx （用于配置路由）

## 安装 JDK 8
